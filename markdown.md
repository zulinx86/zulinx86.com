# Table
```
| Syntax      | Description |
| ----------- | ----------- |
| Header      | Title       |
| Paragraph   | Text        |

<table>
	<tr>
		<th>A</th>
		<th>B</th>
		<th>C</th>
	</tr>
	<tr>
		<td>A-1</td>
		<td>B-1</td>
		<td>C-1</td>
	</tr>
	<tr>
		<td colspan="2">A-2,B-2</td>
		<td>C-2</td>
	</tr>
</table>
```
